
# Tarefa 3.1(b) – Prototipagem  
## Programação Web Avançada – Animalec

Este repositório contém os artefactos produzidos na Tarefa 3.1(b), incluindo:

## Estrutura do Repositório

```
Tarefa_3.1b/
├── wireframes/
├── mockups/
├── prototype/
└── README.md
```

### wireframes/
Contém os wireframes em baixa fidelidade (desktop e mobile) para as entidades:  
- Sponsors  
- Experts  

### mockups/
Contém os mockups (versão visual) também nas versões desktop e mobile.

### prototype/
Inclui o protótipo funcional em HTML5, integrando as duas novas entidades na Página de Administração.

## Objetivo
Aplicar conceitos de prototipagem (wireframe, mockup, protótipo) na extensão da plataforma Animalec, adicionando as entidades **Sponsors** e **Experts**, respeitando os princípios de design responsivo.

## Como visualizar o protótipo
Abrir o ficheiro:

```
prototype/prototipo_animalec.html
```

num navegador (Chrome, Firefox, Edge).
